# Skills — Low Frequency

Skills in this directory have been demoted from active status due to:
- Low usage (not invoked in 30+ days)
- Declining success rate
- Reduced relevance to current objectives

## Review

Low-frequency skills are reviewed every 60 days:
- If `skill_score ≥ 0.70` and recently invoked → re-promote to `active/`
- If still unused after 90 days → deprecate to `deprecated/`

## Criteria
- Not invoked in 30 days → demote to here
- Not invoked in 90 days → deprecate
