# 语音系统

> 7 TTS + 6 STT 提供商的语音能力

## 概述

语音系统提供文本转语音（TTS）和语音转文本（STT/Transcription）能力，支持多种提供商从免费本地方案到高级云端服务。

## TTS (`tools/tts_tool.py`)

### 7 个提供商

| 提供商 | 费用 | 要求 | 字符限制 |
|--------|------|------|---------|
| Edge TTS | 免费（默认） | 无 | 5000 |
| ElevenLabs | 付费 | `ELEVENLABS_API_KEY` | 模型相关 (≤40000) |
| OpenAI TTS | 付费 | `OPENAI_API_KEY` | 4096 |
| MiniMax TTS | 付费 | `MINIMAX_API_KEY` | — |
| Mistral Voxtral | 付费 | `MISTRAL_API_KEY` | — |
| Google Gemini TTS | 付费 | `GEMINI_API_KEY` | 30 预置声音 |
| NeuTTS | 免费 | `neutts_cli` (本地) | — |

### 输出格式

- **Opus (.ogg)** — Telegram 语音气泡
- **MP3** — 其他所有平台

## STT (`tools/transcription_tools.py`)

### 6 个提供商

| 提供商 | 费用 | 特点 |
|--------|------|------|
| local (faster-whisper) | 免费（默认） | 自动下载模型 (~150MB base) |
| groq | 免费层 | Groq Whisper API |
| openai | 付费 | OpenAI Whisper API |
| mistral | 付费 | Mistral Voxtral Transcribe |
| xai | 付费 | 21 语言 + 说话人分离 |

支持格式: mp3, mp4, mpeg, mpga, m4a, wav, webm, ogg, aac, flac
最大文件: 25MB

## 配置项

```yaml
# config.yaml
tts:
  provider: edge  # edge / elevenlabs / openai / minimax / mistral / google / neutts
  voice: ...
  speed: 1.0
```
