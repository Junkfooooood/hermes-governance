# 配置系统

> 运行时配置的分层合并与环境变量展开

## 概述

配置系统通过 `config.yaml` + 环境变量实现灵活的运行时配置，支持深层合并、环境变量引用展开和缓存。

## 核心架构

### 配置层次（优先级从低到高）

```
DEFAULT_CONFIG (硬编码默认值)
    ↓ 合并
~/.hermes/config.yaml (用户配置)
    ↓ 合并
环境变量 (最高优先级)
```

### `load_config()` (line 3281)

1. 加载 `DEFAULT_CONFIG` 和用户 `config.yaml`
2. `_deep_merge()` 递归合并用户配置覆盖默认值
3. 规范化 model key
4. `_expand_env_vars()` 展开 `${VAR}` 引用
5. 缓存展开后的结果

### `save_config()` (line 3381)

- 保留 env-ref 模板（`${VAR}` 不被展开）
- 为未配置的功能添加注释段（security、fallback）

## 实现原理

### DEFAULT_CONFIG 完整 Schema (line 346)

```yaml
model: "model-name"          # 或 dict: {default, fallback}
providers: {}                 # 提供商特定配置
fallback_providers: []        # 降级提供商有序列表
credential_pool_strategies: {} # 每提供商的凭证池策略
toolsets: ["hermes-cli"]      # 启用的工具集

agent:
  max_turns: 90
  gateway_timeout: 1800
  api_max_retries: 3
  tool_use_enforcement: auto  # auto/always/never
  service_tier: null

terminal:
  backend: local              # local/docker/modal/ssh/singularity/daytona
  timeout: 180
  persistent_shell: true
  # Docker/Modal/SSH 子配置...

browser:
  inactivity_timeout: 300
  command_timeout: 60
  record_sessions: false
  allow_private_urls: false
  cdp_url: null

checkpoints:
  enabled: true
  max_snapshots: 50
  auto_prune: true

web:
  backend: firecrawl          # firecrawl/parallel/tavily/exa

context:
  engine: compressor          # 压缩引擎
  # 压缩阈值配置...

memory:
  provider: null              # 记忆插件
  mode: null

security:
  redact_secrets: true
  tirith_enabled: false

fallback_model:
  provider: null
  model: null

custom_providers: []          # 自定义 OpenAI 兼容端点
```

### 环境变量展开

配置值中可使用 `${VAR}` 语法引用环境变量：

```yaml
providers:
  openrouter:
    api_key: ${OPENROUTER_API_KEY}
```

`_expand_env_vars()` 递归遍历所有字符串值并替换。

## 配置项详解

| 配置路径 | 类型 | 默认值 | 说明 |
|---------|------|--------|------|
| `model` | str/dict | — | 模型名或 {default, fallback} |
| `agent.max_turns` | int | 90 | 最大对话轮次 |
| `agent.api_max_retries` | int | 3 | API 重试次数 |
| `terminal.backend` | str | local | 终端后端 |
| `terminal.timeout` | int | 180 | 命令超时(秒) |
| `context.engine` | str | compressor | 压缩引擎 |
| `memory.provider` | str | null | 记忆插件名 |
| `security.redact_secrets` | bool | true | 秘密脱敏 |
