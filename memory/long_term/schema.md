# Memory object schema

## Standard fields
- id
- type
- layer
- importance
- stability
- tags
- source
- timestamp
- ttl
- status
- pointer
- content_summary

## Example frontmatter
---
id: mem_20260428_001
type: investment_thesis
layer: long_term
tags: [finance, valuation]
importance: 0.87
stability: 0.76
status: active
pointer: fact_store/finance/valuation_principles.md
---

## Lifecycle states
- working
- short_term
- long_term
- core
- archived
- deprecated
